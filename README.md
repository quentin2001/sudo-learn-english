## 📖 Overview

This repository holds the material I use to **re‑ignite my English studies**.  It contains:
- **Learning logs** (`ESLPOD.md`) – daily summaries, difficulty ratings, and AI‑driven suggestions for deeper practice.
- **Speaking notes** (`speak.md`) – transcripts, pronunciation checkpoints and role‑play scenarios.
- **Utility scripts** (future) – quick‑run helpers for generating flashcards or audio snippets.

The goal is to keep everything **searchable, version‑controlled, and ready for AI augmentation**.


## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/quentin2001/sudo-learn-english.git
   cd sudo-learn-english
   ```
2. Open any `*.md` file with your favourite Markdown editor (VS Code, typora, etc.).
3. Follow the **AI‑enhanced prompts** inside `ESLPOD.md` to generate custom practice tasks.
